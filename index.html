How to get your code

  1. Open the Discord server.
  2. Find the verify command or verify channel.
  3. Type /verify.
  4. Discord will give you a code.
  5. Copy that code.
  6. Come back to this website.
  7. Paste the code into the box.
  8. Press Verify.
